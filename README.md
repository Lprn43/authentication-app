# authentication-app
This project is a two factor authentication simulation written in C language. The user is expected to meet certain mathematical conditions first, then enter the correct password.
Features
The application basically consists of the following two stages:

Mathematical Verification: The user is asked to enter two integers.
Conditions: The product of two numbers must be greater than 100 (x * y > 100) and their sum must be less than 50 (x + y < 50).

Password Verification: Users who pass the mathematical stage are asked for their password.
Fixed Password: Login123
Users have a total of 3 attempts.
